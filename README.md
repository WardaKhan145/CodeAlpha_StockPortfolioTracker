# Stock Portfolio Tracker
A simple, interactive application designed to calculate total investment values based on user-defined stock quantities and preconfigured stock prices. 
## Project Purpose
The primary objective of this tool is to provide a clean interface where a user can input their stock holdings and instantly view their total financial portfolio valuation. It focuses entirely on internal memory and logic.
## Core Features
* Preconfigured Valuation: Uses a built-in pricing index to safely manage and reference stock market values without requiring external internet connections.
* Interactive Interface: Dynamically prompts users to type in their stock tickers and specify how many shares they own.
* Real-Time Calculations: Multiplies stock quantities by their designated prices instantly, accumulating them into a final grand total.
* Zero File Dependencies: Operates strictly within temporary system memory, eliminating the creation of local text files or spreadsheet exports to ensure a lightweight footprint.
## Practical Logic Applied
Following the design framework of the original task assignment, this project is built around fundamental programming principles:
* Structured Data Mapping for organizing ticker symbols and fixed prices.
* Input and Output Streams for communication between the system and the user.
* Basic Arithmetic for multiplying and aggregating stock values.
